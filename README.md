# VSTool - AI Design Tool

VSTool is an intuitive tool for designing AI models and automatically generating TensorFlow code with a single click.

---

## Key Features

- **Ctrl + Click**: Select nodes.
- **Shift + Click**: Connect nodes.
- **GenerateTF Button**: Automatically generates TensorFlow code from your AI design and copies it to the clipboard.
- **hongjunghun Button**: Opens the developer's GitHub.
- **ongjunghun Button**: Opens the developer's GitHub.

---

## How to Use

1. **Select Nodes**
   - Hold `Ctrl` and click a node to select it.

2. **Connect Nodes**
   - Hold `Shift` and click on nodes to connect them with a line.

3. **Generate TensorFlow Code**
   - After completing your design, click **GenerateTF**.
   - The TensorFlow code will be generated automatically and copied to your clipboard.

4. **Visit GitHub**
   - Click **hongjunghun** to go to the GitHub repository: [GitHub](https://github.com/hongjunghun)
